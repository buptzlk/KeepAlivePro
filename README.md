# KeepAlivePro
## 2022年最强保活黑科技-支持Android4.0-Android12
不同于一像素保活、双进程保活等弱鸡保活手段，该方案是修改frameword层实现的超强保活方案，支持全部Android4.0-Android12， 支持开机自启动，广播事件侦听拉活
